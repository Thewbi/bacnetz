For the cockpit to talk to the simulator, the following prerequisits have to be met:
  1. The bacapp for IP has to be started on the cockpit.
  2. In the HDM a configuration for the bacapp for IP controller has to be present.
  3. The bacnet / IP configuration has to use the same net_number on both links
  4. The bacnet / IP configuration has to use the same net_number as the BACnet devices it is supposed to talk to (Same network)
  5. Make sure the IP traffic is not blocked by any firewall
